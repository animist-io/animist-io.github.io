Whale Island

An Ethereum node running on a micro-computer that connects to mobile clients via Bluetooth server and beacon. Implements an API that targets contract contingencies about location.

Services include:
  • Detection/authentication of contract participants.
  • Endpoints for contract deployment and execution.
  • Endpoints for blockchain queries and payments.
  • A mechanism for contracts to broadcast Bluetooth signal.

    • Wowshuxkluh

    Angular/Ionic module with basic services for Ethereum account management and a suite of methods for connecting to Animist Bluetooth endpoints.

    Features include:
  • Contract compilation and deployment.
  • Contract acquisition for non-deploying parties.
  • Proximity triggered contract execution.
  • Backgrounded / killed-state app wake ups.

    • Wallowa

    A repository of Solidity contracts written around Animist proximity events. A set of basic patterns and functions developers can adapt to generate their own location-based contracts.

    Also contains:
  • Full Solidity unit tests written with Truffle.
  • Continuous integration on Travis CI
  • Documentation for contract APIs

    • Wolves

    A fully decentralized example application for iOS and Android that lets contestants place stakes on a race from one Animist node to another and distributes rewards to the winner.

    A working demo of how to:
  • Authenticate participant presence at a location.
  • Emit app-specific BLE signal by contract on command.
  • Subscribe to contracts someone else has deployed.
  • Set up proximity triggered contract outcomes.
