Components

Whale Island

An Ethereum node on a Rasberry Pi which seamlessly connects to nearby Animist mobile clients over Bluetooth BLE. A local Node.js server intermediates the mobile client and Ethereum, authenticating events and registering them with relevant contracts on the blockchain.

Wowshuxkluh

Javascript APIs for IOS/Android which let apps link an Ethereum account to Animist IOT endpoints. This allows developers to build seamless location-contingent value transfers into any mobile application. Additional modules enable contract semantics that incorporate sensor data collected from the mobile device.

Wallowa

An API for Solidity that lets contracts be written around Animist proximity events and handles the mechanics of publishing these to the blockchain.

Wolves

An online tool to help translate natural language contract ideas into code that can be executed by Animist components on Ethereum. A Web UI that helps people build arbitrarily complex event specifications for real space and time that result in a contractual value exchange. A place to register and display Animist IOT assets and define the cost of using them.
