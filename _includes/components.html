Components

Whale Island

An Ethereum node on a Rasberry Pi which broadcasts a persistant BLE beacon signal. This allows it to launch and connect to nearby Animist mobile clients even if they aren't currently running on their host device. A local Node.js server intermediates the mobile client and Ethereum, authenticating events and registering them with relevant contracts on the blockchain.

Wowshuxkluh

Javascript libraries for IOS and Android which let apps link an Ethereum account to IOT platforms that implement an Animist API. This allows developers to build seamless location-contingent value transfers into any mobile application. Additional modules enable contract semantics that incorporate sensor data collected from the mobile device.

Wallowa

Animist as an oracle on Ethereum: an API for Solidity that lets contracts be written around Animist proximity events and handles the mechanics of publishing these to the blockchain.

Wolves

An online tool to help translate natural language contract ideas into code that can be executed by Animist components on Ethereum itself. A Web UI that helps people build arbitrarily complex event specifications for real space and time that result in a contractual value exchange. A place to register and display Animist IOT assets and define the cost of using them.
