Components

Whale Island

An Ethereum node running on a Rasberry Pi which emits a persistant BLE beacon signal and can automatically launch and connect to nearby Animist mobile clients. This component authenticates location events and can register them with relevant contracts on the blockchain.

An ethereum node running on a wifi connected Rasberry Pi which emits a persistant iBeacon signal and can automatically launch and connect to nearby Animist mobile clients. This component authenticates location events and registers them with relevant contracts on the blockchain.
