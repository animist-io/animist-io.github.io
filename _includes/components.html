Whale Island

An Ethereum node running on a micro-computer coupled with a Bluetooth server/beacon that connects to mobile clients through an Animist API. Services include:

  • Detection/authentication of contract participants.
  • Endpoints for contract deployment and execution.
  • Endpoints for blockchain queries and payments.
  • A mechanism for contracts to broadcast Bluetooth signal.

    • Wowshuxkluh

    Angular/Ionic module with basic services for Ethereum account management and a suite of methods for connecting to an Animist Bluetooth server. Features include:

  • Contract compilation and deployment.
  • Contract acquisition for non-deploying parties.
  • Proximity triggered contract execution.
  • Backgrounded / killed-state app wake ups.

    • Wallowa

    An API for Solidity that lets contracts be written around Animist proximity events and handles the mechanics of publishing these to the blockchain.

    Wolves

    An online tool to help translate natural language contract ideas into code that can be executed by Animist components on Ethereum. A Web UI that helps people build arbitrarily complex event specifications for real space and time that result in a contractual value exchange. A place to register and display Animist IOT assets and define the cost of using them.
